CodePush – Web-like Release Agility for Apps

Ionic makes building mobile apps as easy and fun as Web development - we can use the JavaScript technologies we love and iterate fast using live reload. But while this makes for a great dev experience, the release cycle still leaves much to be desired! Unlike simply pushing code to a web server, it may take weeks to ship new features or bug fixes via the approval process of third party stores, and we cannot guarantee that all our users update their apps diligently.

CodePush fixes this gap by letting you roll out app improvements to end-user devices in a matter of seconds. Combined with other services like VSTS and HockeyApp, this talk will show you how easy it can be to set up a build pipeline, release continuous app deployments and monitor app health, achieving the same web-like agility for mobile projects.

Geoffrey Goh

Geoffrey Goh is a software engineer at Microsoft in Redmond where he works on CodePush.
