Project Ace

Project Ace is a new, open-source Cordova plugin that enables you to easily add native UI and native code to your mobile app. This talk examines the various mechanisms for mixing and matching native UI with HTML, and how to invoke native code. Adam is also looking for feedback on ways to make the combination of Ionic and Ace as productive as possible.

Adam Nathan

Adam Nathan is a developer in Microsoft's developer division and the creator of Project Ace. Over the years, he has written a dozen books on UI technologies, such as the award-winning WPF Unleashed. At Microsoft, he has helped start a number of incubation projects ranging from introducing programming to kids, improving the performance and scalability of speech recognition, and improving .NET interoperability.
