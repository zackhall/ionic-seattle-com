Ionic Framework

Ionic is a complete open-source SDK for hybrid mobile app development. Built on top of AngularJS and Apache Cordova, Ionic provides tools and services for developing hybrid mobile apps using Web technologies like CSS, HTML5, and Sass. In this Meetup, Joshua Hutt will provide an introduction to Ionic and help you get started with it.

Joshua Hutt

Joshua Hutt is a Senior Software Engineer at MaxSam partners. By day, he designs and builds scalable apps in the AWS cloud. By night, he builds SoLoMo apps with Ionic. He loves teaching and sharing the joy of JavaScript.
